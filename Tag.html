"!DOCTYPE html" Tag untuk deklarasi jenis dokumen menjadi HTML

"html" Tag pembuka dan penutup dokumen HTML

"body" Tag pembuka dan penutup isi konten Web

"h1 s/d h6" Tag Pembuat Judul dan Heading

"p" Tag Pembuat Paragraf

"hr" tag untuk membuat garis pembatas


"br" Tag Pembuat garis baru

"style" tag untuk mengatur posisi/style paragraf

"b" tag untuk teks tebal

"i" Tag untuk teks miring

"sub" Tag untuk subscript

"sup" Tag untuk superscript

"strong" Tag untuk Teks Penting

"em" Tag untuk membuat Teks yang ditekankan

"mark" Tag untuk menandai teks

"small" Tag untuk membuat teks kecil

"del" Tag untuk mencoret Teks

"ins" Tag untuk menambahkan garis dibawah teks

"Big" Tag untuk membuat text berhuruf besar
"center" Tag untuk membuat posisi teks menjadi ditengah

"table" Tag untuk membuat table

"Caption" Tag untuk membuat caption tabel
"th" Tag untuk membuat judul/header pada kolom tabel
"td" Tag untuk membuat kolom pada tabel

"form" Tag untuk membuat formulir

"input" Tag untuk Membuat tipe inputan pada form yang akan dibuat

"datalist" Tag untuk Membuat daftar pilihan untuk input data
